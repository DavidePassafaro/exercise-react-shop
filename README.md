# How to run the application

To run the application you need to open two different terminal instances and run the following scripts (one for each instance):

1. npm run dev
2. npm run server

In the "dev" terminal you'll find the localhost url to access the application.<br/>
In the "server" terminal you'll find the url to access pocketbase.
